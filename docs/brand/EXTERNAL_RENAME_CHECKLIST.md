# External Rename Checklist

These items are outside this repository change and must not block UX3 prototype review unless a release owner explicitly makes them blocking.

- GitHub repository and remote URL rename.
- Website and domain identity updates.
- Store, Steam, or other distribution listing identity.
- Code-signing subject and certificate identity.
- Package registry publication names.
- Release URL and download path updates.
- CI secret, badge, artifact, and cache key names.
- Installer publisher metadata.
- Social profile, avatar, header, and card assets.
- Trademark clearance and legal review.

The included Setpiece identity kit is production-shaped design source material, not trademark clearance.
