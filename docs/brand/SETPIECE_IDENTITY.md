# Setpiece Identity

Setpiece is the product name.

Brand definition: A carefully arranged, repeatable sequence of coordinated actions.

Tagline: Everything in place. Ready to run.

Use `ritual` as the domain noun for a multi-step attended procedure:

- Run a ritual.
- Ritual picker.
- Active ritual.
- Ritual Suggestions.

Do not use Setpiece as a replacement noun for ritual procedures. Setpiece is the app and local command surface.

The canonical UI expression for UX3 is Setpiece Paper: warm paper surfaces, graphite text, mineral actions, and semantic state colors from `brand/setpiece-v1/Setpiece_Identity_Kit_v1/06_Guidelines/setpiece_design_tokens.json`.

Tray state identity uses the bundled Setpiece state icons for ready, running, waiting, confirmation, failure, recovery, paused, and stopped. The tooltip remains authoritative, and stopped is supplemental only; normal idle returns to ready after a run is recorded.

The identity kit is source material, not legal trademark clearance.
